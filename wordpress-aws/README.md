# Configuring a WordPress website on AWS EC2 with MariaDB database

## Objective
The goal of this lab was to deploy a fully functional WordPress website on AWS using:
1. **EC2 Instance (Amazon Linux 2023)**
2. **Apache HTTP Server**
3. **PHP**
4. **MariaDB Database**
5. **phpMyAdmin**

The lab also required proper folder structure, file permissions, and database user setup to pass the validation checks.
