# AWS VPC Deployment Using CloudFormation 

## 📌 Project Overview

This project demonstrates how to provision and manage a Virtual Private Cloud (VPC) using Infrastructure as Code (IaC) with AWS CloudFormation.

The goal is to design a scalable and structured network architecture by deploying and updating a VPC using predefined CloudFormation templates stored in Amazon S3.

This project reflects real-world cloud engineering practices where infrastructure is automated, version-controlled, and easily reproducible.
